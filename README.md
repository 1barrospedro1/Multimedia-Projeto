# Multimedia-Projeto

* Aluno: Pedro Barros | Nº 33248
* Tema: Escape Room

O jogo é uma experiência de terror e escape room em primeira pessoa. O jogador encontra-se preso num pesadelo e precisa de explorar o ambiente, recolher ferramentas e resolver puzzles para destrancar a porta de saída antes que o tempo acabe. Se conseguir escapar da primeira sala, o jogador entra na fase final: uma cena de perseguição intensa onde terá de escapar e fugir do pesadelo.

## Funcionalidades
* **Controlador do Jogador:** Andamento, corrida, agachamento e salto fluídos utilizando o motor de física do Unity.
* **Visão com o rato:** Matemática de rotação de câmera customizada que ignora a interpolação de física para evitar micro-travamentos (micro-stutters).
* **Lanterna:** Uma lanterna que pode ser ativada ou desativada com um efeito 'flicker' personalizado baseado em corrotinas.
* **Movimento Dinâmico da Cabeça ('Head Bobbing'):** Movimento da câmera que simula naturalmente os passos de caminhada/corrida (e para quando o jogador está no ar).
* **Interação:** Mecânica para interagir com objetos.

## Controlos
* **W, A, S, D** - Mover
* **Mouse** - Olhar
* **Spacebar** - Saltar
* **Left Shift** - Correr
* **Ctrl** - Agachar
* **F** - Ligar/Desligar lanterna
* **E** - Interagir
